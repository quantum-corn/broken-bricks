This series of blogs is about a project called Latexipy that I am working on. This is about creating a Python module to bring LaTeX into Python. Just to give you some idea of what I have in mind, I want to able to create LaTeX documents faster than originally workig on LaTeX, and use the benefits of a programming language, that is to say, ability perform arithmetic, symbolic algebra, and plots etc. to add features to document typesetting that pure LaTeX doesn't offer. Come ride the waves with me as I build it up.
