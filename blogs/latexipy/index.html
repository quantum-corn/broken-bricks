I just started teaching high school physics online. And in the very first class I came to realize that I need to write mathematical equations and to draw diagrams and the whiteboard on any online video conferencing platform wasn't suited to do that. Yes, using a graphic tablet helps, but I didn't have one and I didn't want to buy one either. So, I was looking for alternatives.

With the fact that I have worked with LaTeX quite extensively before, that was the first thing to come to my mind. But a few moments of considering it, and I realized, it's not going to be fast enough to work as I speak. I needed something that could add similar blocks of certain commands on the fly, with the option to modify parameters, macros, so to speak. Much like a programming laguage functions. Now LaTeX allows creating macros as far as I know, but I never used it before and I wasn't feeling up for learning because LaTeX cannot do mathematical processing anyway, whereas a full programming language would add that benefit. So I decided to create some interface in some programming language.

I chose Python over anything else becuase:

  • It has some existing LaTeX support that I know of. With a Google search I figured it was Pylatex. But it was done really granular and I didn't find the interface fast enough, or maybe I judged it too quick, I just didn't want to put in the time t learn it, I guess.
  • Python has great support for doing science and math. So that combined with my interface would be a great help. Among these I was certainly looking at a certain Python package called Sympy. It's a fabulous thing that let's you carry out symbolic algebra within Python and also can give the output as LaTeX.
And since I was creating a tool, I thought it might be good to create it like an importable package. Now that is one thing I never did in all my time with Python, so I set out to learn about that before I came back to the project. That part of the story can be found here.

So this project Latexipy, is about a creating a tool in Python that adds intuitiveness and speed to working with LaTeX.
